# MaxFlowMinCut
Projenin amacı, azami akış (maximum flow) olarak geçen ve düğümler (nodes) arasında akış kapasiteleri belirli bir şekildeki (graph) bir başlangıçtan bir hedefe en fazla akışın sağlanmasının ardından akışın sistemden geçmemesi için literatürde min-cut olarak geçen yöntemin uygulamanmasıdır.
